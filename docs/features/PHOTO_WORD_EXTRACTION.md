# Распознавание слов с фото

## Описание

Функция позволяет пользователю добавлять слова, сфотографировав страницу с текстом или списком слов. Изображение отправляется в GPT-4o-mini с поддержкой vision, который извлекает все слова на изучаемом языке, игнорируя слова на родном языке.

## Пользовательский сценарий

1. Пользователь открывает страницу редактирования колоды (DeckEditorPage)
2. Рядом с полем ввода слов нажимает кнопку "Сканировать с фото"
3. На мобильном телефоне открывается камера (или выбор файла на десктопе)
4. Пользователь фотографирует страницу с текстом
5. Открывается диалог с превью фото и индикатором загрузки
6. LLM извлекает слова на изучаемом языке из фото
7. Извлечённые слова отображаются как чипсы с кнопкой удаления (X)
8. Пользователь удаляет ненужные слова (например, уже знакомые)
9. Нажимает "Добавить слова" — слова попадают в WordChipsInput
10. Далее работает обычный пайплайн: автоперевод, настройка медиа, генерация карточек

## Техническая реализация

### Backend

**Эндпоинт:** `POST /api/cards/extract-words-from-photo/`

**Файлы:**
- `backend/apps/cards/llm_utils.py` — функция `extract_words_from_photo()`
- `backend/apps/cards/views.py` — вью `extract_words_from_photo_view`
- `backend/apps/cards/urls.py` — маршрут в `analysis_urlpatterns`
- `backend/apps/cards/token_utils.py` — константа `PHOTO_OCR_COST = 1`

**LLM:** GPT-4o-mini с vision (`detail: "high"`)

**Промпт:** Извлечь все слова на изучаемом языке, игнорировать родной язык, вернуть JSON `{"words": [...]}`. Температура 0.1 для детерминированного результата.

**Обработка ошибок:** При ошибке LLM токены автоматически возвращаются (refund).

### Frontend

**Файлы:**
- `frontend/src/components/PhotoWordExtractor.tsx` — компонент с камерой и диалогом
- `frontend/src/components/SmartWordInput.tsx` — интеграция компонента
- `frontend/src/services/deck.service.ts` — метод `extractWordsFromPhoto()`
- `frontend/src/lib/api-constants.ts` — эндпоинт
- `frontend/src/locales/*.ts` — ключи локализации (8 языков)

**Камера:** `<input type="file" accept="image/*" capture="environment">` — на мобильном открывает заднюю камеру.

## Стоимость

1 токен за операцию (аналогично генерации изображения или аудио).

## Ограничения

- Максимальный размер фото: 10 МБ
- Поддерживаемые форматы: JPEG, PNG
- Качество распознавания зависит от чёткости фото и разборчивости текста
- Рукописный текст может распознаваться хуже печатного
- LLM может возвращать словоформы вместо базовых форм (для немецкого работает `processGermanWords`)
