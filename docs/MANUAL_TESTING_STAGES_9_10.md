## Раздел 2: Генерация изображений с разными стилями

### Тест MT-004: Минималистичный стиль
- [ ] **Шаг 1:** Выбрать стиль "Минималистичный"
- [ ] **Шаг 2:** Добавить немецкое слово (например: "Haus")
- [ ] **Шаг 3:** Ввести перевод (например: "дом")
- [ ] **Шаг 4:** Сгенерировать изображение
- [ ] **Шаг 5:** Визуально оценить изображение
- [ ] **Ожидаемый результат:** 
  - Изображение простое, чистое, минималистичное
  - Один основной объект, без лишних деталей
  - Светлый фон
  - Нет текста, букв, цифр и символов
- [ ] **Результат:** ___________

### Тест MT-005: Сбалансированный стиль
- [ ] **Шаг 1:** Выбрать стиль "Сбалансированный"
- [ ] **Шаг 2:** Добавить немецкое слово (например: "Buch")
- [ ] **Шаг 3:** Ввести перевод (например: "книга")
- [ ] **Шаг 4:** Сгенерировать изображение
- [ ] **Шаг 5:** Визуально оценить изображение
- [ ] **Ожидаемый результат:** 
  - Выразительная иллюстрация, ясно передающая смысл
  - Нейтральный стиль, умеренно детализированный
  - Естественные цвета
  - Светлый или ненавязчивый фон
  - Нет текста, букв, цифр и символов
- [ ] **Результат:** ___________

### Тест MT-006: Творческий стиль
- [ ] **Шаг 1:** Выбрать стиль "Творческий"
- [ ] **Шаг 2:** Добавить немецкое слово (например: "Liebe")
- [ ] **Шаг 3:** Ввести перевод (например: "любовь")
- [ ] **Шаг 4:** Сгенерировать изображение
- [ ] **Шаг 5:** Визуально оценить изображение
- [ ] **Ожидаемый результат:** 
  - Яркая, сочная иллюстрация
  - Разнообразная цветовая палитра
  - Красивый свет, приятные детали
  - Может быть минимализм, рисованный арт, живопись, 3D или реалистичность
  - Нет текста, букв, цифр и символов
- [ ] **Результат:** ___________

### Тест MT-007: Сравнение стилей для одного слова
- [ ] **Шаг 1:** Добавить немецкое слово (например: "Baum")
- [ ] **Шаг 2:** Ввести перевод (например: "дерево")
- [ ] **Шаг 3:** Сгенерировать изображение в минималистичном стиле
- [ ] **Шаг 4:** Сгенерировать изображение в сбалансированном стиле
- [ ] **Шаг 5:** Сгенерировать изображение в творческом стиле
- [ ] **Шаг 6:** Сравнить три изображения
- [ ] **Ожидаемый результат:** 
  - Три изображения визуально отличаются друг от друга
  - Минималистичный - самый простой
  - Творческий - самый детализированный и яркий
  - Сбалансированный - средний по деталям
- [ ] **Результат:** ___________
