# Чек-лист тестирования этапа 10

**Дата создания:** 17 декабря 2024  
**Этап:** 10 (Режимы генерации изображений)

## Общая информация

Этот чек-лист предназначен для комплексного тестирования функциональности, реализованной в этапе 10:
- **Этап 10:** Режимы генерации изображений (минималистичный, сбалансированный, творческий)

## Автоматические тесты

Автоматические тесты находятся в:
- **Backend:** `backend/apps/cards/tests.py` (классы `TestDefaultPrompts`)

**Запуск автоматических тестов:**

**Быстрый запуск (рекомендуется):**
```bash
cd backend
pytest apps/cards/tests.py::TestDefaultPrompts -v
```

**Ручной запуск:**
```bash
# Backend тесты
cd backend
pytest apps/cards/tests.py::TestDefaultPrompts -v

# Frontend тесты (если есть)
cd frontend
npm test -- --watchAll=false
```

## Чек-лист функциональности

### 1. Интерфейс выбора стиля

- [ ] Компонент выбора стиля отображается на главной странице
- [ ] Видны три пиктограммы (Минималистичный, Сбалансированный, Творческий)
- [ ] По умолчанию выбран стиль "Сбалансированный"
- [ ] При клике на пиктограмму стиль меняется
- [ ] Выбранный стиль визуально выделяется

### 2. Генерация изображений с разными стилями

- [ ] Минималистичный стиль генерирует простые, чистые изображения
- [ ] Сбалансированный стиль генерирует выразительные иллюстрации
- [ ] Творческий стиль генерирует яркие, сочные иллюстрации
- [ ] Все стили не содержат текста, букв, цифр и символов
- [ ] Изображения для одного слова отличаются в зависимости от стиля

### 3. Применение стиля для колоды

- [ ] Выбранный стиль применяется ко всем изображениям в колоде
- [ ] При изменении стиля новые изображения генерируются с новым стилем
- [ ] Стиль передается в API при генерации колоды

### 4. Интеграция с генерацией колоды

- [ ] Параметр `image_style` передается в запрос `/api/cards/generate/`
- [ ] Все изображения в колоде имеют одинаковый стиль
- [ ] Колода успешно генерируется и скачивается
